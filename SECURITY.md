# Security Policy

## Supported Versions

Security updates are provided for the **latest major release** of this project,
and for the **previous major release** if it remains widely used.

All other versions are not guaranteed to receive security updates.

A list of releases for this project can be found here:
https://github.com/D4isDAVID/dxm/releases

## Reporting a Vulnerability

If you believe you have found a security vulnerability, please report it
**privately** using one of the following channels:
- GitHub: https://github.com/D4isDAVID/dxm/security/advisories/new
- Email: malchin459@gmail.com
- Discord: https://discord.com/users/258650797171015680

When reporting a vulnerability, please provide as much detail as possible to
help us investigate, such as:
- A comprehensive description of the vulnerability and its potential impact.
- Steps to reproduce the issue (if applicable).
- The affected version(s) of the project.
